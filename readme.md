
#glibc
GLibc（The GNU C Library ），是GNU发布的libc库，即c运行库。glibc是linux系统（fish:not kernel）中最底层的api，几乎其它任何运行库都会依赖于glibc。glibc除了封装linux操作系统所提供的系统服务外，它本身也提供了许多其它一些必要功能服务的实现。
